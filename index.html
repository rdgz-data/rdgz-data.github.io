Predictive Maintenance with Industrial Sensors

Predictive maintenance project developed using XGBoost and industrial sensor data. The model achieves 92.1% accuracy and 92.4% recall in detecting failures, providing a robust solution to anticipate machine breakdowns, reduce downtime, and optimize maintenance schedules. Results are documented in GitHub and a Tableau dashboard will be added for interactive visualization.

This project applies advanced data mining techniques to predict electoral variables in Spain, integrating territorial, demographic, and socioeconomic information at both provincial and regional levels. Through a complete workflow that includes data cleaning, feature engineering, variable selection, cross-validation, and predictor-importance analysis, the model identifies meaningful patterns in electoral behavior and provides a robust, reproducible, and adaptable methodological framework for understanding complex territorial dynamics.
